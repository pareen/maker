# Maker Portfolio

A portfolio platform for makers to showcase everything they've built - from side projects to startups.

## Features

- **User Authentication**: Sign up and login with email/password
- **Profile Management**: Edit your bio, domains, social links, and first make story
- **Project Tracking**: Add and manage projects with stages from Idea to IPO
- **Stage Progression**: Visual representation of project progress through 10 stages
- **Role Classification**: Track your role (Solo, Co-founder, Early team, Contributor)
- **Public Profiles**: Shareable profile pages with stats and project history
- **Social Sharing**: Share your profile on Twitter, LinkedIn, Facebook, WhatsApp, and Email

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Local Storage for data persistence
